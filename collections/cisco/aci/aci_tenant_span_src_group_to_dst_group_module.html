cisco.aci.aci_tenant_span_src_group_to_dst_group – Bind SPAN source groups to destination groups (span:SpanLbl)¶

Note

This plugin is part of the cisco.aci collection (version 2.0.0).

To install it use: ansible-galaxy collection install cisco.aci.

To use it in a playbook, specify: cisco.aci.aci_tenant_span_src_group_to_dst_group.

  • Synopsis

  • Parameters

  • Notes

  • See Also

  • Examples

  • Return Values

Synopsis¶

  • Bind SPAN source groups to associated destination groups on Cisco ACI fabrics.

Notes¶

Note

  • The tenant, src_group, and dst_group must exist before using this module in your playbook. The cisco.aci.aci_tenant, cisco.aci.aci_tenant_span_src_group, and cisco.aci.aci_tenant_span_dst_group modules can be used for this.

Examples¶

- cisco.aci.aci_tenant_span_src_group_to_dst_group:
    host: apic
    username: admin
    password: SomeSecretPassword
    tenant: production
    src_group: "{{ src_group }}"
    dst_group: "{{ dst_group }}"
    description: "{{ description }}"
  delegate_to: localhost
